# 介绍
本项目是自用的一个为XXT评教的脚本。
---

## 说明
- python版本是3.12.10
- 使用浏览器的后台评教对此时的我尚属复杂，而且也没有太多必要性，在几次尝试后选择使用图形化界面操作来实现。
- 用学习通PC端打开评教界面选中一个界面开始，保守估计的评教效率是2分钟5条。
- 评教格式是五个满分20的分值填空，一个主观评价填空。
- 这个代码是基于我以前写的一个不能稳定运行的脚本使用trae多次粗放微调写成的，不甚优美也情有可原。
- tes是Teaching Evaluation Script的缩写，没大写是为了输入方便。
- 退出只要切换界面等着就好了，其他操作无效。

---

## 使用说明
这里推荐使用终端：
- 先打开终端，将地址设置到脚本的根目录下；
- python+空格是使用电脑的python环境，但为了准确，可以改成用从系统根目录一直到python.exe的的路径代替python。
- 例如：cd 脚本.py根目录（enter）D:\python31210\python.exe 脚本.py（enter）
